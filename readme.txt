solucao desafio eventos

A arquitetura usada foi MVC, model view controller por ser adequada e suficiente para o app proposto, onde podem se organizar bem as camadas e folders do código seperando suas responsabilidades e ficar assim também de fácil entendimento para outros desenvolvedores.
As bibliotecas usada são:
Retrofit para consumo dos WS.
Gson e Moshi para parse das respostas dos WS.
Butterknife (em uma classe, para demonstrar seu uso)
Picasso para load das imagens.
OSM (open street map) para localizar o pin de lat e long no mapa na tela detalhes.
E demais bibliotecas do Android como RecuclerView, CardView, etc.
